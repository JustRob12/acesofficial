# ACES Website

Modern website for Association of Computing and Engineering Students built with Next.js, TypeScript, and Tailwind CSS.

## 🚀 Deploy to Vercel

1. **Push to GitHub**
   ```bash
   git add .
   git commit -m "Initial commit"
   git push origin main
   ```

2. **Deploy to Vercel**
   - Go to [vercel.com](https://vercel.com)
   - Sign in with GitHub
   - Click "New Project"
   - Import your repository
   - Click "Deploy"

## 🛠️ Tech Stack
- Next.js 14
- TypeScript
- Tailwind CSS
- Vercel (Deployment)

## 📁 Structure
- `/public/events/` - Event images
- `/public/logos/` - Organization logos  
- `/public/officers/` - Officer photos
- `/src/app/` - Next.js app router pages

- Responsive design
